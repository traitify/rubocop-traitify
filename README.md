# RuboCop Traitify

Traitify's default configuration for [RuboCop](https://github.com/rubocop-hq/rubocop).

## Installation

```
gem "rubocop-traitify"
```

## Usage

Add the following to `.rubocop.yml`

```
inherit_gem:
  rubocop-traitify: .rubocop_traitify.yml
```
